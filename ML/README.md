# 🤖 Machine Learning Projects

This folder contains machine learning-related projects and Jupyter notebooks focused on data science, model development, and analysis, completed during the Cellula Technologies Internship.

## 📁 Projects Included

- **`Booking Cancellation Prediction/`**: A project focused on predicting whether a booking will be canceled using machine learning classification models. Includes a detailed Jupyter Notebook, dataset, presentation materials, and a deployable Flask web application.
- **`Trip Fare Prediction/`**: A project aimed at predicting trip fares using regression models and analyzing important factors that influence pricing. Includes a comprehensive Jupyter Notebook, trained models (including Neural Networks), and a complete Flask web application for interactive predictions.

## 🛠️ Technologies Used
- Python
- Jupyter Notebooks
- Scikit-Learn
- Pandas & NumPy
- Flask
