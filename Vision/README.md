# 👁️ Computer Vision Projects

This folder contains computer vision projects and notebooks exploring image processing, object detection, and visual analysis techniques, completed during the Cellula Technologies Internship.

## 📁 Projects Included

- **`Teeth images Classification/`**: A computer vision project focused on classifying images of teeth using deep learning techniques. Includes project reports, training notebooks, trained models, and an interactive Streamlit application.
- **`Water Segmentation/`**: An image segmentation project aimed at detecting and segmenting water bodies within images. Includes training notebooks, trained models, and a deployable Flask web application.
- **[Shoplifting Detection Project](https://github.com/MahmoudRajab77/Shoplifting_Detection_Project)** (Teammate's Profile): A comprehensive video classification system for detecting shoplifting activities from surveillance footage. The system utilizes multiple deep learning architectures (3D CNN, CNN+RNN, Transformer, and Pretrained R3D) to classify videos into "shoplifting" and "non-shoplifting" categories.

## 🛠️ Technologies Used
- Python
- Jupyter Notebooks
- OpenCV
- TensorFlow / Keras
- Flask
- Streamlit
